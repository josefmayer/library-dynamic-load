## Library Dynamic Load
Dynamic loading of a shared library. <br />
Calling a function from library.



### Technologies
Dynamic library, dynamic loader, Linux



### Steps
Compile program: <br />
*make progdl* <br />

Run program: <br />
*./progdl* <br />

Access program from CLI: <br />
Enter: library method argument <br />
> libm.so cosf 0.0
